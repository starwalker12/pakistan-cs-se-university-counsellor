# University Admission Chatbot - Project Report

## Phase 1: Project Setup

This phase involved setting up the project structure, creating starter files,
initializing Git, and pushing to GitHub.

### File Structure

```
university-admission-chatbot/
├── index.html          # Main HTML page
├── style.css           # Styling for the chat interface
├── script.js           # Chatbot logic (FAQ search + Ollama integration)
├── data/
│   └── faq.json        # FAQ data for the chatbot
├── docs/
│   ├── report.md       # Project report (this file)
│   └── presentation-outline.md  # Presentation outline
└── README.md           # Project overview and deployment notes
```

### Technologies Used
- HTML
- CSS
- JavaScript
- JSON
- Ollama (local AI model)
- Git / GitHub
