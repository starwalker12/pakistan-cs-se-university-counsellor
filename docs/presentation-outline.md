# University Admission Chatbot - Presentation Outline

## Slide 1: Title
- Project Name: University Admission Chatbot
- Your Name
- Course / Date

## Slide 2: Problem Statement
- Students have many questions about admissions
- Repetitive queries for university staff
- Need for a simple, 24/7 available assistant

## Slide 3: Solution
- AI-powered chatbot
- Answers admission-related questions
- Works with local AI (Ollama) or FAQ fallback

## Slide 4: Technologies
- HTML, CSS, JavaScript
- JSON for FAQ data
- Ollama with Gemma model for AI responses
- Deployed on Vercel

## Slide 5: How It Works
1. User types a question
2. JavaScript searches FAQ for matching keywords
3. Sends context to Ollama for AI-generated response
4. Falls back to FAQ if Ollama is unavailable

## Slide 6: Demo
- Show the chatbot in action
- Ask a question that matches FAQ
- Ask a question that uses AI

## Slide 7: Deployment
- Hosted on Vercel for free
- FAQ fallback works without local setup
- Ollama required only for AI features

## Slide 8: Conclusion
- Simple, effective solution
- Easy to maintain and extend
- Thank you! Questions?
